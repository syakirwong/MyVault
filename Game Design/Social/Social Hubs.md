# Social Hubs

---

## Hub Design Philosophy

Hubs are where builds are SEEN. In a text-based interface, this means player lists with visible build summaries, name plates with earned cosmetics, and interactive build inspection.

---

## Major Hubs

### Port Haven (Calm Shores)

- **Function:** Starter hub, trade center
- **UI Features:** NPC Marketplace panel, crafting interface, commission board, guild recruitment board
- **Social Elements:** Player list with build previews, trading post, dueling queue
- **Capacity:** 200 players in player list

### Sky Forum (Celestial Peaks)

- **Function:** Endgame social hub
- **UI Features:** Creator's Hall (build showcase feed), Arena queue, Legendary crafter storefronts
- **Social Elements:** Build display board (top builds visible), guild hall directory, featured builds
- **Capacity:** 300 players in player list

### Thunder Arena (Stormreach)

- **Function:** PvP hub
- **UI Features:** Arena queue panel, PvP leaderboards, honor vendor
- **Social Elements:** Match results feed, winner spotlights, spectator mode (live combat log)
- **Capacity:** 150 players in player list

---

## Social Features in Hubs

- **Inspect Player:** View any player's build card — gems, alignment, achievements, stats
- **Build Display Mode:** Pin your build card to the player list for others to inspect
- **Emote Commands:** Text-based emotes visible in hub chat
- **NPC Commission Board:** Browse crafter services, submit gem orders
- **Guild Recruitment Board:** Searchable guild listings with requirements and descriptions
- **Dueling Queue:** Opt-in PvP from hub, results posted to hub feed

---

## Hub Events

- **Weekly:** Featured build showcase (top creator's build displayed on build board)
- **Monthly:** Trade festival (reduced NPC marketplace fees, boosted crafting)
- **Seasonal:** Arena tournament with spectator mode (live combat logs)
- **Special:** Developer Q&A events in Sky Forum chat

# Alignment System

Alignment defines a player's philosophical stance and directly modifies combat, visuals, gem behavior, and available content. It is not cosmetic. It is a core progression axis.

## Three Alignments

### Radiant (Order / Light)

- **Philosophy:** Protection, stability, community.
- **Combat Focus:** Support, shields, sustained healing, area denial.
- **UI Language:** Gold text, white accents, clean borders, upward-facing symbols.
- **Gem Modifications:** Skills gain protective/supportive secondary effects.
- **Trade-off:** Lower burst damage. Abilities favor sustained encounters.
- **Exclusive Content:** Radiant sanctuaries, holy dungeons, protection quests.

### Corrupted (Chaos / Void)

- **Philosophy:** Power at any cost, risk-reward.
- **Combat Focus:** High burst, lifesteal, DoTs, self-damage mechanics.
- **UI Language:** Purple text, crimson accents, jagged borders, distortion symbols.
- **Gem Modifications:** Skills gain damage/risk secondary effects.
- **Trade-off:** Self-damage, instability, shorter buff durations.
- **Exclusive Content:** Void rifts, corruption dungeons, power quests.

### Unbound (Balance / Neutrality)

- **Philosophy:** Flexibility, adaptation, hybrid mastery.
- **Combat Focus:** Fast cooldowns, hybrid offense/defense, skill cycling.
- **UI Language:** Silver text, pale blue accents, smooth borders, balanced symbols.
- **Gem Modifications:** Skills gain utility/flexibility secondary effects.
- **Trade-off:** No extreme power spikes. Master of none.
- **Exclusive Content:** Balance trials, neutral faction missions, diplomacy quests.

## Alignment Progression

Alignment is measured on a scale from 0 (neutral start) to 100 (deep alignment).

Points are earned through:

- Story choices
- PvP behavior (mercy vs execution)
- Quest alignment
- Gem usage patterns
- Guild alignment contribution

## Depth Tiers

| Depth | Tier | Unlock |
|---|---|---|
| 0-20 | Initiate | Basic alignment name color |
| 21-40 | Devoted | Alignment-modified gem effects |
| 41-60 | Zealot | Exclusive alignment gems (2) |
| 61-80 | Ascendant | Alignment badge upgrade + build card border |
| 81-100 | Paragon | Title prefix + signature combat log style + alignment mastery skill |

## Switching Cost

Switching alignment is possible but costly. Cost scales with depth.

| Depth | Penalty |
|---|---|
| 0-20 | Free switch, no penalty |
| 21-40 | Lose 50% alignment progress |
| 41-60 | Lose all alignment progress + 1 random gem loses 2 mastery levels |
| 61-80 | Above + destroy one alignment-exclusive gem |
| 81-100 | Above + 48-hour "Unstable" debuff (reduced stats) |

The escalating cost is deliberate. Early exploration is free. Deep commitment carries weight.

## Alignment Interactions

- **Radiant vs Corrupted:** Bonus damage in PvP (10%), narrative tension.
- **Unbound:** Neutral to both. Can trade with all factions.
- **Guild Alignment:** Guilds can declare alignment, unlocking guild-wide bonuses.
- **Mixed-Alignment Parties:** Viable but miss alignment synergy bonuses.
